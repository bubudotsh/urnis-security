
# Epitool

Auto compliation and norm check


## Installation

git clone this repo

```bash
  ./install.sh
```
    
this scipt add an alias in your bashrc and zshrc (if your have)
## Usage/Examples

```bash
    Options : 
        -t      loop time in second 
        -h      Help

    Usage :
        epitool -t 5
```

## Update

```bash
  epitool -u
```
